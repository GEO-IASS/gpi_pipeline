GPI Data Pipeline User’s Guide¶

This chapter describes how to use the GPI data pipeline software to reduce and calibrate data from GPI.

See also the IFS Data Handbook, which describes the characteristics of GPI IFS data in terms of hardware performance, known artifacts, and best practices for scientific analysis.
